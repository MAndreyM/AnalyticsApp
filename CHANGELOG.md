# Changelog

Все заметные изменения в этом проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-11

### Добавлено
- **Phase 1**: Базовая структура проекта с CMake и Ninja
- **Phase 2**: Модели данных и система анализа успеваемости
  - Модели `Student` и `AnalysisResult` с валидацией данных
  - `GradeCalculator` с логикой округления оценок (≥ 0.6 → округление вверх)
  - 4 комплексных тестовых сценария анализа успеваемости
  - Утилиты файловой системы для кроссплатформенных операций
  - Полная система тестирования с doctest

- **Phase 3**: Интеграция обработки файлов
  - Модуль `FileProcessor` для рекурсивного сканирования директорий
  - `DirectoryScanner` с поддержкой статистики и обработкой ошибок
  - `ExcelFileFilter` для фильтрации файлов по расширению .xls
  - Интегрированное главное меню с поддержкой аргументов командной строки
  - Демонстрационные программы для всех компонентов

### Изменено
- Рефакторинг структуры проекта для лучшей модульности
- Улучшена обработка ошибок и валидация данных
- Оптимизирована система сборки с объектными библиотеками

### Технические детали
- C++20 стандарт с поддержкой `<filesystem>`
- Кроссплатформенная сборка (RedOS, Alt Linux, Windows)
- CI/CD пайплайны для автоматического тестирования
- Полная документация Doxygen

## [0.0.1] - 2025-11-09
