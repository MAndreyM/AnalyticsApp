## 🚀 Использование

### Командная строка
```bash
# Интерактивный режим
./analytics_app

# Справка
./analytics_app --help

# Сканирование директории
./analytics_app --scan ./data
```

```cpp

DirectoryScanner scanner;
auto files = scanner.findExcelFiles("/path/to/data");
auto stats = scanner.getLastScanStatistics();
```
